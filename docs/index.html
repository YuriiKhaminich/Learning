Chronicle of my learning

This page contains information about my learning in web development

I started my studies on November 2, 2020 on the professional course of WebCademy online school: Profession: Web developer.

The course includes:

  1. HTML5: valid semantic layout.
  2. CSS3: responsiveness, effects and animations.
  3. Working with layouts in Figma and PhotoShop.
  4. SCSS: preprocessor for easy working with CSS.
  5. Programming basics: JavaScript, jQuery, PHP.
  6. Best layout practices: BEM naming, Flex and Grid CSS.
  7. Bootstrap 4 for fast and convenient website layout.
  8. How the Internet, server and browsers work.
  9. All necessary programs for a layout designer: VS Code, Avocode, Filezilla, PhotoShop and others.

The "span" tag is easily styled by CSS or manipulated with JavaScript using the "class" or "id" attribute. The "span" tag is much like the "div" element, but "div" is a block-level element and "span" is an inline element.
